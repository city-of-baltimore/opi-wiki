# Create Planner Groups Without Automatic Member Alerts

{{ page_header(summary="Short how-to for reducing inbox noise when setting up Microsoft Planner work.") }}

AUDIENCE
:   OPI staff

OWNER
:   Chief of Staff

## Why this guide exists

Creating a Planner plan or assigning tasks can trigger member notifications by
default. This guide captures the quieter setup path described in the source
document.

## Option 1: Create the group in Outlook first

1. Open Outlook on the web or desktop.

2. Go to `Groups` and choose `Create group`.

3. During setup, clear the option that sends copies of group conversations and
   events to member inboxes.

4. Add members after the group is created.

5. Use that group when you create or connect the Planner plan.

## Option 2: Use Power Automate for silent task assignment

If direct assignment in Planner creates too much email noise, use Power Automate
to build a workflow that assigns tasks without the default Planner alerts.

## When to use which path

- Use the Outlook group setup when you are creating a new plan and want a lower
  notification baseline from the start.

- Use Power Automate when the plan already exists or when you need more control
  over how tasks are assigned.
