# Open Data

Open Data is OPI's standing program for making city data public. Through Open Baltimore, the City publishes datasets openly so residents, Council, journalists, researchers, and partners can verify what the City reports and build on its data. The program is required by [Baltimore City Code, Article 1, Subtitle 9](https://codes.baltimorecity.gov/us/md/cities/baltimore/code/1/9).

## What the program does

The program publishes and maintains high-quality open datasets on Open Baltimore. It sets the publishing standards, covering plain language, accessibility, metadata, and refresh cadence, that keep open data current and usable. It applies the classification tiers so only data cleared for public release is published, and precise or sensitive fields are generalized or withheld. And it feeds the [Citywide Data Network](../citywide-data-network/index.md), giving partners and residents something to build on.
