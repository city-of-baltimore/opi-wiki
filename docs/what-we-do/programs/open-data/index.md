# Open Data

> Baltimore's program for publishing city data openly — so residents, Council, journalists, and researchers can see and use it.

Open Data is OPI's standing program for making city data public. Through **Open Baltimore**, the City publishes datasets openly so residents, Council, journalists, researchers, and partners can verify what the City reports and build on its data. The program is required by [Baltimore City Code, Article 1, Subtitle 9](https://codes.baltimorecity.gov/us/md/cities/baltimore/code/1/9).

## What the program does

- Publishes and maintains high-quality open datasets on **Open Baltimore**.
- Sets publishing standards — plain language, accessibility, metadata, and refresh cadence — so open data stays current and usable.
- Applies the classification tiers so only data cleared for public release is published, and precise or sensitive fields are generalized or withheld.
- Feeds the [Citywide Data Network](../citywide-data-network/index.md), giving partners and residents something to build on.
